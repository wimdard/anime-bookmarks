# Anime Bookmarks

Простое приложение для хранения аниме-закладок

Запуск:

1. Скачайте проект с GitHub.
2. Распакуйте архив.
3. Откройте файл `index.html` в браузере.

Все закладки автоматически сохраняются локально в браузере и остаются после повторного запуска приложения.
(если вы удалите данные файлы, то все что вы сохранили, будет все удаленно)
